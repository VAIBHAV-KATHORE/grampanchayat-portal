# Grampanchayat Portal

A web-based Grampanchayat management system developed using PHP and MySQL.  
This project helps citizens and administrators manage complaints, notices, and village information online.

---

## 🚀 Features

- User Registration and Login
- Admin Dashboard
- Complaint Management System
- Notice Board
- Citizen Information System

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP Server

---

## 📦 Libraries Used

- TCPDF (for PDF generation)

---
